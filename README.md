# Games Store

Landing page estática para una tienda de videojuegos. El proyecto muestra un encabezado principal, una sección de juegos populares, una grilla de productos, un bloque de suscripción y un pie de página. Está construido solo con HTML y CSS, así que se puede abrir directamente en el navegador sin instalar dependencias.

## Características

- Diseño responsive para escritorio y móvil.
- Menú de navegación colapsable en pantallas pequeñas.
- Secciones visuales para destacar juegos populares y catálogo.
- Estilo visual oscuro con acentos naranjas.
- Formulario de suscripción de ejemplo.

## Tecnologías

- HTML5
- CSS3
- Google Fonts (`Poppins`)

## Estructura del proyecto

- `index.html`: estructura principal de la página.
- `style.css`: estilos y adaptación responsive.
- `images/`: imágenes usadas en el hero, menú y catálogo.

## Cómo ejecutarlo

1. Abre la carpeta del proyecto en VS Code o en tu explorador de archivos.
2. Abre `index.html` en tu navegador.
3. Si prefieres un servidor local, usa la extensión Live Server o cualquier servidor estático.

## Personalización rápida

- Cambia textos, precios y enlaces dentro de `index.html`.
- Sustituye las imágenes en `images/` por tus propios recursos.
- Ajusta colores, tipografía y espaciados desde `style.css`.

## Notas

- El formulario no envía datos a un backend.
- Los enlaces del menú y los botones son de ejemplo y apuntan a `#`.

## Vista previa

La página está pensada como una portada comercial para mostrar productos destacados y llevar al usuario hacia una compra o contacto posterior.
